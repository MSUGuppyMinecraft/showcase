Loading Your Game

We've sorted download files below which should allow you to continue work on your world and check out your teammate's worlds. Importing the games will depend on what version you have.

Minecraft Education
This is the version we used to build the game. It is generally only available to educators, but you may have a license through your school.
Download the Education Edition file and double click the file or use the Import button in-game.
Minecraft Bedrock Edition on PC
Minecraft Education does not officially support conversion to Bedrock but we used a third-party tool to convert the maps and they appear to be fully-intact even including features that are normally exclusive to Education Edition (like NPCs).
Download the Bedrock file and double click the file or use the Import button in-game.
Minecraft Bedrock Edition on Console or Mobile
Importing the map to console or mobile can be done, but it's kind of complicated.
  1. You (or a friend) will need a copy of Bedrock Edition on PC.
  2. Import the file to PC following the instructions above.
  3. Click the edit button next to the imported world.
  4. Click Play on Realms. This will create a Minecraft Realms Server (An official Minecraft Service that provides server hosting for a monthly fee - currently starting at $4 a month. If you have never used it before, you should have access to a free trial.)
  5. Invite the account on your console or mobile device to the realm.
  6. Click the edit button next to the Realm on the console or mobile device and select Download.
Minecraft Java Edition
This is the original version of Minecraft, which does not receive the same updates as the Bedrock Edition and is less compatibile.
It is possible to convert the file using a 3rd-party tool called Chunker, (which is not an official Minecraft product, but is generally trusted by the community.) We attempted to convert some of the files with Chunker, but they were missing a lot of the essential elements that are exclusive to Bedrock and Education Edition and added some strange structures due to misreading map heights, so we have not included those. You are welcome to attempt to convert the files yourself, but they may still require a lot of changes before they're playable again.
